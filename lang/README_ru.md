Расширение VS Code для управления и сбора отмеченных файлами заметок. Организуйте свои заметки эффективно через чистый и интуитивный интерфейс.

## Возможности

- **Организация заметок по тегам** : Классифицировать заметки в пользовательские теги с поддержкой многоуровневых тегов каталогов
- **Полнотекстовый поиск** : Мгновенный поиск содержимого во всех файлах заметок
- **Управление тегами** : Легко создавать, переименовывать и удалять теги
- **Импорт/Экспорт** : Импортировать файлы резервных копий JSON для восстановления данных, экспортировать коллекцию в файлы резервных копий TXT или JSON
- **Перетаскивание** : Перетаскивать файлы/папки заметок в разные теги для быстрой классификации
- **Интеграция файлов** : Открывать заметки в новых окнах или показывать в проводнике файлов
- **Включение/Выключение заметок** : Переключать видимость заметок без их удаления
- **Поддержка нескольких языков** : Поддерживает 12 языков
  - English, 简体中文, 繁體中文, Français, Deutsch, Españол
  - 日本語, 한к국어, Русский, Italiano, Português (Brasil), Türkçe, Polski, Češtина

## Установка

### Установка из VS Code Marketplace

1. Открыть VS Code
2. Перейти к панели расширений (Ctrl+Shift+X)
3. Поиск "Note Collection" или "коллекция заметок" или [установить из Marketplace](https://marketplace.visualstudio.com/items?itemName=SaraKale.note-collection)
4. Нажать "Установить"

![](../image/extensions-1.png)

### Установка из файла VSIX

1. Скачать последний файл `.vsix` со страницы [Releases](https://github.com/sarakale/vscode-note-collection/releases)
2. Открыть VS Code
3. Нажать Ctrl+Shift+P, чтобы открыть палитру команд
4. Выбрать "Extensions: Install from VSIX..."
5. Выбрать загруженный файл `.vsix`

![](../image/extensions-2.png)

## Использование

### Начало работы

1. После установки представление "Note Collection" появится в левой панели активности
2. Нажать на иконку, чтобы открыть боковую панель
3. Начинайте добавлять свои заметки!
4. Необходимо перезапустить VS Code, чтобы сменить язык.

### Основные операции

#### Добавление заметок в теги

- Нажмите правой кнопкой мыши на тег, выберите "Импорт файлов/папок..." чтобы добавить файлы/папки заметок

![](../image/menu-1.png)

- Перетащите файлы/папки из проводника файлов в теги

![](../image/path-2.gif)

- Если файл перемещен или удален, появится предупреждающий значок и сообщение.

![](../image/fileerror.png)

- Вы также можете вручную вводить пути в дополнительных операциях, что откроет выделенную панель Webview для ввода большего количества путей.
    - Ввод тега, вы можете ввести несколько тегов, разделенных английскими запятыми:
        ```
        Note1,Note2
        ```
    - Ввод полного пути к файлу, по одному пути к файлу на строку:
        ```
        D:\path\file1.txt
        D:\path\file2.txt
        ```
		
![](../image/path-1.png)

#### Управление тегами

- Нажмите "Добавить теги..." в меню "Ещё", чтобы создать новые теги/многоуровневые теги
- Нажмите правой кнопкой мыши на теги, чтобы добавить теги, переименовать теги, удалить теги
- Удаление тега не удалит файлы заметок, которые он содержит, только удалит их из коллекции
- Нажмите на значок сворачивания, чтобы развернуть/свернуть всё содержимое тегов
- Вы также можете переместить теги внутрь других тегов

![](../image/tag-1.png)

![](../image/tag-2.gif)

![](../image/tag-3.gif)

#### Поиск заметок
- Нажмите на значок поиска на панели инструментов
- Введите ключевые слова поиска, чтобы найти заметки во всех тегах
- Отображается до 50 совпадающих результатов
- Поддерживает только общие текстовые форматы; изображения/документы/видео будут искать только имена файлов. Для двоичных файлов стороннее программное обеспечение для полнотекстового поиска, такое как: Recoll, DocFetcher и др.

![](../image/search-1.gif)

#### Операции с заметками
- **Открыть** : Дважды щелкнуть или выбрать "Открыть файл" из контекстного меню
- **Показать в проводнике** : Открыть расположение файла в проводнике
- **Переименовать заметку** : Можно изменить на новое имя
- **Редактировать теги** : Добавлять несколько тегов к заметкам
- **Удалить заметку** : Удалить заметки из коллекции
- **Включить/Выключить** : Скрыть/показать элементы файлов заметок, не удаляя файл.

![](../image/menu-2.png)

### Экспорт TXT / Импорт Экспорт файлов резервных копий JSON

- В дополнительных операциях:
- **Экспорт в TXT** : Экспортировать весь список коллекции заметок в текстовый файл
- **Экспорт файла резервной копии JSON** : Создать файл резервной копии JSON для сохранения состояния вашей коллекции заметок для легкого восстановления или миграции на другие устройства.
- **Импорт файла резервной копии JSON** : Восстановить состояние коллекции заметок из ранее экспортированного файла резервной копии JSON.
  - Примечание: Импорт файла резервной копии JSON перезапишет текущее состояние коллекции заметок, действуйте с осторожностью.

## Известные проблемы

1. После первой установки вам может потребоваться перезапустить VS Code, чтобы сменить язык.
2. При сворачивании/разворачивании тегов могут возникнуть проблемы обновления пользовательского интерфейса. Попробуйте вручную развернуть теги для решения.

## Инструкции по сборке

### Локальная разработка

1. Клонировать репозиторий
```bash
git clone https://github.com/sarakale/vscode-note-collection.git
cd vscode-note-collection
```

2. Установить зависимости
```bash
npm install
```

3. Скомпилировать TypeScript
```bash
npm run compile
```

4. Нажмите F5 в VS Code, чтобы начать отладку

### Упаковка расширения

1. Установить vsce (VS Code Extension Packager)
```bash
npm install -g vsce
```

2. Упаковать в файл `.vsix`
```bash
npm run package
```

Или использовать команду vsce напрямую
```bash
vsce package
```

3. Сгенерированный файл `.vsix` можно установить вручную.
