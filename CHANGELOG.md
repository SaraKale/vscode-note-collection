## [1.0.0] - 2026-02-28

### Added
- Initial release of Note Collection extension
- **Core Features**:
  - Note management with tag support
  - Create, rename, and delete tags
  - Import note files into tags
  - Export note collection as TXT
  - Backup and restore functionality
  - Full-text search across all notes
  - Drag and drop support for organizing notes
  - Enable/disable notes without deletion
  - Open notes in new windows or in file explorer
  - Edit tags for individual notes

- **UI/UX**:
  - Clean sidebar view in Activity Bar
  - Intuitive toolbar with quick access actions
  - Context menus for note and tag operations
  - Collapsible tag groups

- **Multi-language Support** (12 languages):
  - English (en)
  - 简体中文 (zh-cn)
  - 繁體中文 (zh-tw)
  - Français (fr)
  - Deutsch (de)
  - Español (es)
  - 日本語 (ja)
  - 한국어 (ko)
  - Русский (ru)
  - Italiano (it)
  - Português (Brasil) (pt-br)
  - Türkçe (tr)
  - Polski (pl)
  - Čeština (cs)

### Technical Details
- Built with TypeScript
